Como proyecto final para la universidad, desarrollé una herramienta de **gestión de inventario** con el propósito de **optimizar** las operaciones del negocio Goicoechea. Este sistema se diseñó para mejorar el **monitoreo de stock**, los registros de empleados y, en consecuencia, aumentar las ventas.

El desafío clave fue crear una **base de datos robusta** capaz de manejar un inventario complejo. Lo logré diseñando un **sistema de metadatos dinámicos** que permite definir atributos personalizados para cada producto. También durante el proceso, adquirí experiencia en la construcción de Single-Page Applications (SPA) que operan en **tiempo real**.
