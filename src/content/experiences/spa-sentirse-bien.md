Desarrollé una aplicación web integral como proyecto final para una materia universitaria. Esta web sirve como la **presentación principal** de los **servicios** de un Spa ficticio, permitiendo a los clientes **reservar** y **pagar** citas de manera directa.

La aplicación también incluye un módulo específico para empleados, donde pueden **visualizar** y **gestionar** los turnos pendientes, lo que **optimiza la organización interna** del negocio. El proyecto demuestra mi habilidad para construir **soluciones web completas** que atienden tanto las necesidades de los usuarios finales como la gestión interna de una empresa.
